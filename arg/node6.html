Drum patterns

Drum patterns are expressed as 16 bit strings for each of the 8 drum components, resulting in a 128 bit string. Each 16 bit sub-string divides the time in 16 beats, a bit set to 1 represents that the drum component is played at that particular time and a bit set to 0 represents that it is mute.
